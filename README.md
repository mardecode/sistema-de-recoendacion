# sistema-de-recoendacion
Sistema de recomendación algoritmo KNN basado en grafos 
## Requerimientos
- Pistache C++
- Rapidjson
- C++ 17

## Compilación
make

##Ejecutar 
make run

Abrir en el navegador: 
http://localhost:8090

idItem,idUser,rating,timestamp
